### Проект: Компоненты Bitrix.

**Список компонентов:**

**buttons_back_and_up:** Выводит ссылки "Назад" и "Наверх" (навигация по логической структуре сайта).

**steam_widget_group:** Выводит виджет группы Steam.

**twitch:** Встраивает канал Twitch. При указании ключа и идентификатора клиента получает дополнительные данные от Twitch API.

**youtube:** Встраивает видео из плэйлиста Youtube. Режимы работы: статический и AJAX.
